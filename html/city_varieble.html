city_varieble

Description

This function is used for city variables derivation.

Usage

city_varieble(df = df, city_cols = NULL, city_pattern = "city$",
  city_class = city_class, parallel = TRUE)

Arguments

df

A data.frame.

city_cols

Variables of city,

city_pattern

Regular expressions, used to match city variable names. Default is "city$".

city_class

Class or levels of cities.

parallel

Logical, parallel computing. Default is TRUE.
