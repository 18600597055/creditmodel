date_cut {creditmodel}R Documentation

Date Time Cut Point

Description

date_cut is a small function to get date point.

Usage

date_cut(dat_time, pct = 0.7)

Arguments

dat_time

time vectors.

pct

the percent of cutting. Default: 0.7.

Value

A Date.

Examples

date_cut(dat_time = lendingclub$issue_d, pct = 0.8)
