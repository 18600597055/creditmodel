pred_score {creditmodel}R Documentation

pred_score

Description

pred_score is for using trained models to predict new data.

Usage

pred_score(model, dat, x_list = NULL, bins_table = NULL,
  obs_id = NULL, sp_values = list(-1, "Missing"), save_data = TRUE,
  woe_name = TRUE)

Arguments

model

Model generated by training_model.

dat

Dataframe of new data.

x_list

Into the model variables.

bins_table

a data.frame generated by get_bins_table

obs_id

The name of ID of observations or key variable of data. Default is NULL.

sp_values

Special values.

save_data

Logical, save the result or not. Default is FALSE.

woe_name

Logical. Whether woe variable's name contains 'woe'.Default is TRUE.

Value

new scores.

See Also

training_model, lr_params, xgb_params, rf_params
